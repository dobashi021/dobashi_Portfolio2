# 生成AIで手順書自動生成PoCを構築

**目的：** QA手順書作成工数の削減と標準化を目指す。  
**アプローチ：**
- 定型文＋キーワード入力によりAIが自動で手順書を生成  
- Python Flask＋GPT APIでプロトタイプを作成  

**成果：**
- 平均工数を60％削減  
- 現場レビュー工数も短縮  

**今後の展望：**
- AI出力の品質評価（自動レビュー）機能の追加  
- MLOps／AI品質検証への拡張
